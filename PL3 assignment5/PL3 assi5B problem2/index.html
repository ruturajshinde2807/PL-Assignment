Maximum/Minimum Element in Array

Problem Statement :

Write a program in any language to find the maximum and minimum element in an array.

Sample Code in C: 
       
#include < stdio.h >
#include < conio.h >
 
 
int main()
{
    int a[1000],i,n,min,max;
   
    printf("Enter size of the array : ");
    scanf("%d",&n);
 
    printf("Enter elements in array : ");
    for(i=0; i< n; i++)
    {
        scanf(" %d",&a[i]);
    }
 
    min=max=a[0];
    for(i=1; i< n; i++)
    {
         if(min>a[i])
		  min=a[i];   
		   if(max< a[i])
		    max=a[i];       
    }
     printf("minimum of array is : %d",min);
          printf("\nmaximum of array is : %d",max);
 
 
    return 0;
}
Sample Input :
4
50 12 36 53

Sample Output :
minimum of array is : 12
maximum of array is : 53

variable Used :
int a[1000],i,n,min,max are the variable used for these code

Time Complexity :
O(n)
